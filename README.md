# COVID-19 Impact Analysis in Pakistan
COVID-19 data analysis and dashboard using Python and Power BI

Project Overview
This project analyzes the spread and impact of COVID-19 in Pakistan using confirmed cases, deaths, recoveries, and travel history data.

Tools Used
- Python (pandas)
- Power BI
- Excel / CSV

Key Analysis
- Overall COVID situation (Cases, Deaths, Recovery Rate)
- Time-based trends
- Province-wise and city-wise impact
- Travel history analysis
- Peak day identification

Project Structure
- data/ – cleaned datasets
- python/ – pandas analysis
- powerbi/ – Power BI dashboard
- report/ – project report

Key Insights
- COVID cases peaked during mid-2020
- Punjab and Sindh were the most affected provinces
- Karachi and Lahore were major hotspots
- Recovery rate improved over time
